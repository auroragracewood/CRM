How it works

A plug-in is a single .py file with a NAME constant plus any of the known hooks as top-level functions (on_contact_created, on_deal_stage_changed, compute_fit_score, etc.). The host catches plug-in exceptions and writes the traceback to plugins.last_error — a broken plug-in cannot crash the server. See agent_surface/plugins/README.md for the full hook list and the example_fit_score.py walkthrough.
