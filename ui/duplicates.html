Each group lists contacts the system thinks might be the same person. Detection: exact-email match (after normalization), normalized name + same company, similar-name (Levenshtein) with lower confidence. Check 2 or more in a group — the first checked wins; the rest are merged into it. Interactions, notes, tags, deals, tasks all move to the winner; the loser's email frees up. Losers are soft- deleted, so a merge is recoverable from /contacts?show_deleted=1.
