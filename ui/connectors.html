Create an inbound endpoint

External systems POST JSON to /in/<slug>. Sign the body with HMAC-SHA256 using the endpoint's shared secret and pass it as X-CRM-Inbound-Signature. The default routing parses email + name from the top-level payload, tags the contact inbound:<slug>, and logs a system interaction on their timeline. Customize via PUT /api/inbound-endpoints/<id>.
